## How to run

1. Clone this repository
   ```sh
   git clone git@github.com:mehedisarkar2k/morning-sparklers.git
   ```
2. Run
   ```sh
   yarn
   ```
3. Then run to see project in live server
   ```sh
   yarn start
   ```
4. This app should be available in http://localhost:3000/
